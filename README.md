# java-cli-maven-failsafe-testng-test-hello-world

## Description
A POC for maven app using testng
framework with failsafe plugin.

## Tech stack
- java
- maven
  - testng
  - failsafe

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
